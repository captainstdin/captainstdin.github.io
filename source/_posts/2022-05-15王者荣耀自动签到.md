### 3. 环境变量设置，和定时器设置

```json
{
    "triggers": [
        {
            "name": "dayOn1clock",
            "type": "timer",
            "config": "0 2 0 * * * *"
        }
    ]
}

```

### 设置4个环境变量
```
gameroleid
msdktoken
token
userid
```
