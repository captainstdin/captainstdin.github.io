读取Linux TTY

<!--more-->

```php
<?php

//echo "1234"; // 5 characters of padding at the end
//    echo "\033[6D"; // Move 5 characters backward
function read(){
    $fp = fopen('/dev/stdin', 'r');
    $input = fgets($fp, 255);
    fclose($fp);
//移除字符串右侧的字符：
    $input = chop($input);
    return $input;
}

echo read();
```
