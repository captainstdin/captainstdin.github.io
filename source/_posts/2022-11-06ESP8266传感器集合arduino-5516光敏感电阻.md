### 模拟元件 【5516光敏电阻模块】（ADC）

核心代码

```c++
#include <Arduino.h>
const int AdcPin = A0;

void setup() {
// write your initialization code here
    Serial.begin(115200);
}

void loop() {
// write your code here
    int adc = analogRead(AdcPin);
    Serial.print("value=");
    Serial.println(adc);
    delay(1000);
}
```
